# Wumpus-Game
A Python-based simulation of the classic Wumpus World AI problem using OOP. It features a $4 \times 4$ grid where an intelligent agent uses sensory cues (Stench, Breeze, Glitter) to navigate, avoid deadly hazards like pits and monsters, and retrieve hidden gold. Focuses on logic-based decision making, state-space search, and modular code design.
